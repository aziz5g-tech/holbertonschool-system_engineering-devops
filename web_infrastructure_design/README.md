# Web Infrastructure Design

This folder contains the answers for the web infrastructure design tasks.

## Files

- `0-simple_web_stack` — one-server web infrastructure for `www.foobar.com`
- `1-distributed_web_infrastructure` — distributed infrastructure with a load balancer and database replication
- `2-secured_and_monitored_web_infrastructure` — secured infrastructure with HTTPS, firewalls, and monitoring
- `3-scale_up` — scaled architecture with separated tiers and a HAproxy cluster

## Purpose

Each file explains the required infrastructure diagram, the role of each component, and the issues or trade-offs mentioned in the task instructions.
